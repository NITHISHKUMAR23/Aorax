# 🌌 Aorax  
### *The Precision Financial Guidance Engine*

---

## 🚀 Overview  

**Aorax** is a high-performance financial engine designed to bridge the gap between **long-term wealth goals** and **daily spending behavior**.

Built on the **Aoraniti Philosophy**, it transforms static monthly budgets into a dynamic **Daily Spend Limit (DSL)** that adapts in real time.

> 💡 Instead of just tracking expenses, Aorax actively **guides financial decisions**.

---

The name **Aoraniti** represents:

- **Aora (Aurora / Dawn)** → Clarity, new beginnings, removal of confusion  
- **Niti (Guidance / Ethics)** → Discipline, structured financial direction  

Together, it means:

> **“Guided financial clarity at the start of every decision.”**

**Aorax** is the mathematical **axis (X)** that powers this philosophy.

## ⚙️ Core Engine: Dual Calculation Model  

Unlike traditional budgeting tools, Aorax runs **two parallel financial strategies**:

---

### 🔹 1. Adaptive Flow (Strict Mode)

This mode enforces discipline by protecting your savings goal.

- Overspending today reduces future limits  
- Ensures goal completion (e.g., ₹1 Lakh target)  

#### Formula:
DSL_adaptive = (Current Balance - (Fixed Expenses + Target Savings)) / Days Remaining


#### Behavior:
- Real-time recalculation  
- Strict financial correction  
- Goal-first approach  

---

### 🔹 2. Rolling Balance (Flexible Mode)

This mode allows flexibility while tracking overspending.

- Keeps a fixed daily limit  
- Maintains a running balance (credit/debit)

#### Formula:
Net Balance = (Standard Limit - Actual Spend) + Previous Balance


#### Behavior:
- Flexible lifestyle support  
- Tracks surplus (🟢) or deficit (🔴)  
- Encourages recovery over time  

---

## 🧠 Core Concept: Financial Buckets  

Aorax enforces strict financial separation:

| Bucket            | Type    | Description |
|------------------|--------|------------|
| Fixed Vault       | 🔒 Locked | Rent, EMI, bills |
| Goal Vault        | 🔒 Locked | Savings target |
| Emergency Vault   | 🔒 Locked | Safety buffer |
| Spending Bucket   | 🔓 Active | Daily usable money |

> ❗ Only the **Spending Bucket** is accessible for daily expenses.
